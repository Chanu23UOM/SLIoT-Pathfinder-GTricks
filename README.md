# 🌲 Pathfinder Smart Post 
**An AI-Powered Wilderness Safety & Information Mesh for SLIoT 2026**

![Badge](link_to_your_generated_badge_image.png)

## 📌 The Problem
In remote adventure tourism, hikers enter "Digital Blind Spots" where 4G/5G fails. Pathfinder provides offline guidance and real-time AI danger detection using frugal engineering.

## 🛠️ Features
* **Offline Captive Portal:** Serves a local trail map, telemetry, and RangerBot directly to smartphones without the internet.
* **Edge AI Seismic Detection:** Utilizes TinyML (via Edge Impulse) to classify elephant footfalls from ground vibrations.
* **LoRa SOS Backbone:** Transmits emergency distress signals over long distances, bypassing cellular networks.

## 📂 Project Structure
* `/src` - The core ESP32 C++ firmware.
* `/data-generation` - Python scripts for creating synthetic HIL (Hardware-in-the-Loop) training data.
* `/docs` - Strategic blueprints and pitch materials.
